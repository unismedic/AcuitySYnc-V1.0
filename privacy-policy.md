🔐 Privacy Policy – AcuitySync Application

Effective Date: [April, 19 2026]

1. Introduction

AcuitySync is a nursing-focused application designed to support clinical handover, acuity assessment, and patient care coordination. We are committed to protecting user privacy and ensuring data security in alignment with healthcare best practices and applicable regulations.

2. Information We Collect
a. User-Provided Information
Clinical inputs entered manually (e.g., patient acuity, handover notes)
Non-identifiable operational data
b. Device & Usage Data
Application usage metrics
Device type, browser type (for performance optimization only)
3. Data Storage & Security
Data is stored locally (cached) within the user’s device/session
No automatic transmission of patient data to external servers
The application is hosted securely on Google Cloud infrastructure
No sensitive patient-identifiable data is intentionally collected or stored externally

We follow best practices aligned with:

Intensive Care Society
I-PASS Handoff Bundle
ISBAR Communication Tool
4. Data Use

The application is intended for:

Clinical decision support
Handover standardization
Quality improvement and auditing

Data entered is used only within the application context and is not sold, shared, or distributed.

5. Data Sharing

We do not:

Share data with third parties
Sell or monetize user or patient data
Store identifiable patient information externally

Future integration with Hospital Information Systems (HIS) may occur under institutional governance and compliance policies.

6. User Responsibilities

Users are responsible for:

Avoiding entry of full patient identifiers unless within a secure institutional environment
Ensuring compliance with their hospital’s data protection policies
7. Security Measures
Local data caching only
Secure cloud deployment environment
No external API exposure of patient data
8. Updates to This Policy

This Privacy Policy may be updated periodically. Users will be notified of significant changes through the application or repository updates.
